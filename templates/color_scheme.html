Application Color Scheme

This page defines the complete color system for your web application using CSS custom properties (variables).

Usage Instructions

Include this color scheme in your application by:

  1. Copy the CSS variables from the :root selector
  2. Add them to your main stylesheet
  3. Use the variables throughout your CSS: color: var(--primary-600);
  4. Toggle dark mode by adding data-theme="dark" to your html element
/* Example usage in your CSS */ .my-button { background-color: var(--primary-600); color: var(--button-primary-text); border: 1px solid var(--border-medium); } .my-button:hover { background-color: var(--primary-700); }
