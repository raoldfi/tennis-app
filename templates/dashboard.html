Getting Started
Recommended Order

For best results, add entries in this order:

  1. Facilities - Tennis venues
  2. Leagues - Competition categories
  3. Teams - Assign to leagues & facilities
  4. Matches - Schedule between teams
Search and Sort

All tables are sortable by clicking column headers. Use browser search (Ctrl+F) to find specific items.

Filter by League

Use the league filters on Teams and Matches pages to view data for specific leagues.

Bulk Import

For large datasets, use the command line interface:

python sqlite_tennis_db.py tennis.db load facilities facilities.yaml
Export Data

View data in JSON format using the command line list commands.

Pro Tip: Start by adding a few facilities and leagues, then build out your teams and schedule matches. Use the USTA Constants page to ensure you're using valid values for league creation. New: Use Calculate Pairings to generate balanced match schedules automatically!
