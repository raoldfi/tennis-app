Import Leagues
Upload a YAML file containing league data. File must have .yaml or .yml extension.
Expected Format
leagues:
  - id: 101
    name: "2025 Adult 18+ 3.0 Women"
    year: 2025
    section: "Southwest"
    region: "Northern New Mexico"
    age_group: "18 & Over"
    division: "3.0 Women"
    num_matches: 6
    num_lines_per_match: 4
    allow_split_lines: false
Important Notes
  • Valid sections: Must be one of the official USTA sections (e.g., Southwest, Pacific Northwest)
  • Valid regions: Must be one of the USTA regions (e.g., Northern New Mexico, Southern California)
  • Valid age groups: 18 & Over, 40 & Over, 55 & Over, 65 & Over
  • Valid divisions: Format like "3.0 Women", "4.5 Men", "7.0 Mixed"
  • Optional fields: start_date, end_date, preferred_days, backup_days
League Import/Export Help
Overview

The import/export functionality allows you to:

  • Import: Add multiple leagues from a YAML file
  • Export: Save league data to YAML or JSON format
  • Backup: Create backups of your league configurations
  • Share: Transfer league setups between databases
YAML Format Example
leagues:
  - id: 101
    name: "2025 Adult 18+ 3.0 Women"
    year: 2025
    section: "Southwest"
    region: "Northern New Mexico"
    age_group: "18 & Over"
    division: "3.0 Women"
    num_matches: 6
    num_lines_per_match: 4
    allow_split_lines: false
    preferred_days: ["Saturday", "Sunday"]
    start_date: "2025-03-01"
    end_date: "2025-05-31"
  - id: 102
    name: "2025 Adult 18+ 4.0 Men"
    year: 2025
    section: "Southwest"
    region: "Northern New Mexico"
    age_group: "18 & Over"
    division: "4.0 Men"
    num_matches: 8
    num_lines_per_match: 3
    allow_split_lines: true
Import Process
  1. Prepare your YAML file
  2. Click "Import YAML"
  3. Select your .yaml/.yml file
  4. Review import status
  5. Refresh page to see results
Export Options
  • Export All to YAML: All leagues
  • Export All to JSON: JSON format
  • Export Single: Individual league
